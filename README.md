# gutentag-world-lua

> Gutentag, World! in Lua

## Run

```
lua gutentag_world.lua
```


## Install

```
luarocks install gutentag-world
```


### Usage

## Usage

```sh
lua gutentag_world.lua
```

Or require in your project:

```lua
local gw = require("gutentag_world")
print(gw.gutentag_world())
```
